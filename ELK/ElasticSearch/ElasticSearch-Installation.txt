wget https://artifacts.elastic.co/downloads/elasticsearch/elasticsearch-8.1.1-linux-x86_64.tar.gz
tar -xzvf elasticsearch8.1.1.tar.gz 

ElasticSearch to bind all IP address
go to elasticsearch/config/elasticsearch.yaml and add the following
 
 network:
  host: _eth0_

 
To run the elastic search 
./bin/elasticsearch

kibana installation

wget https://artifacts.elastic.co/downloads/kibana/kibana-8.1.1-linux-x86_64.tar.gz
tar -xzvf kibana-8.1.1-linux-x86_64.tar.gz

to make kibana accept connection from outside world add the following in 
vi kinbana/congig/kibana.yml and add the following

server.host: "0.0.0.0"

if it want to accessed only from specific IP then add like below

server.host: "13.xx.xx.xx"

To Start Kibana
./bin/kibana
