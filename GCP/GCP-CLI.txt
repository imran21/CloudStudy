Ref:- https://cloud.google.com/storage/docs/configuring-cors#gsutil_2
Setting the CORS orgin for bucket
CORS.json file
[
    {
      "origin": ["https://cms.xl.id.apptium.com", "http://localhost:8080"],
      "method": ["GET"],
      "responseHeader": ["*"],
      "maxAgeSeconds": 3600
    }
]
Then give the following command in gcp cmd cli to set the CORS for bucket
>gsutil cors set E:\KP\axiata-dev-cms-CORS.json gs://axiata-dev-cms

To view the CORS of a bucket
>gsutil cors get gs://axiata-dev-cms

To Remove CORS orgin
CORS.json file
[]
Then enter the same command for setting the CORS
>gsutil cors set E:\KP\CORS.json gs://axiata-dev-cms
